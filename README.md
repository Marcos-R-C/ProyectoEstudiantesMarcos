Para ejecutar la aplicación, debe situarse en la carpeta students-marcos y ejecutar el comando npm run dev en el terminal.

También hay que tener instaladas las dependencias necesarias de React, Tailwind, y Node si no está instalado en el equipo. En mi caso he utilizado Tailwind CLI en vez de

Se han utilizado las clases de Tailwind para dar formato a los distintos componentes de la aplicación, mientras que se ha utilizado React para elaborar la página mediante la división en varios componentes de esta con una mezcla de html y código JavaScript. Se ha incluido la utilización de un useState de React para monitorizar el array de estudiantes, de forma que se actualice la página cuando haya algún cambio de estado en este.
